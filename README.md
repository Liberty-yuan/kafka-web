### kafka-web
---
> kafka-web是使用spring开发的一个kafka状态信息管理的工具系统。通过该系统可以直观地看到kafka中zookeeper/broker/topic的各种状态(包括broker的位置、端口/topic的relication、partitions、leader node等)，同时也可以分页显示每个topic的messages
